This project is a grounded, transformer-based booking assistant capable of handling restaurant, hotel, and train reservations using a hybrid dataset and a service catalog for factual responses.

### Features
- Understands booking-related conversations
- Extracts attributes (area, price, cuisine, stars, time)
- Matches services from a structured catalog
- Generates natural grounded responses using a fine-tuned T5 model
- Confirms bookings and generates reservation reference numbers
- Streamlit interface for real-time interaction
